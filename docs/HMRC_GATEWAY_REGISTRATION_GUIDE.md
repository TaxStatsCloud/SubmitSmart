### **STEP 1: Create Your Government Gateway Account**

1. Visit the official HMRC registration page:
   ```
   https://www.gov.uk/log-in-register-hmrc-online-services
   ```

2. Click **"Create sign in details"** (Organisation accounts section)

3. Select account type:
   - **"Organisation"** if filing for your company
   - **"Agent"** if you're an accountant filing for clients

4. Choose **"Create Government Gateway user ID"**

5. Enter your email address and create a password
   - Password requirements: Minimum 8 characters, mix of letters and numbers

6. Set up your **12-digit User ID**
   - Save this securely - you'll need it for all future logins
   - Example format: 123456789012

7. Choose 2-factor authentication method:
   - SMS text message (recommended)
   - Authentication app
   - Voice call

8. Complete registration and note down your **User ID**

### **STEP 2: Add Corporation Tax Service**

1. Log into your Government Gateway account using your new User ID and password

### **STEP 3: Wait for Activation Code (7-10 Working Days)**

**What happens now:**
- HMRC will **post an activation code** to your company's registered office address
- This is a **security measure** to verify you have authority to file for the company
- The code is valid for **28 days** from the date of issue

**Important Notes:**

### **STEP 4: Activate Corporation Tax Service**

1. Once you receive the postal activation code, log back into Government Gateway

2. Go to **"Your account"** â†’ **"Services you can add"**

3. Find **"Corporation Tax"** in the list

6. Confirm activation

7. **Wait 24-48 hours** before attempting to file
   - HMRC systems need time to fully activate your account
   - Filing too soon may result in authentication errors

## ğŸ”§ Troubleshooting Common Issues

### Error 1046 - Authentication Error

**Possible causes:**
1. âŒ Government Gateway User ID or Password incorrect
   - *Solution:* Verify credentials are exactly as created (case-sensitive)
   - *Reset if needed:* https://www.tax.service.gov.uk/account-recovery/

2. âŒ Corporation Tax service not activated yet
   - *Solution:* Check "Your account" to see if service shows as active
   - *Wait time:* Allow 24-48 hours after activation

3. âŒ UTR entered incorrectly
   - *Solution:* Double-check UTR matches exactly (no spaces)
   - *Find UTR:* Check CT603 notice or HMRC letter

4. âŒ Company linked to agent credentials
   - *Solution:* If switching from accountant to self-filing, contact HMRC to remove agent link
   - *HMRC helpline:* 0300 200 3410

### Code Not Received After 10 Days

**What to do:**
1. Check with your registered office that mail isn't being held
2. Verify HMRC has the correct address on file
3. Call HMRC helpline: **0300 200 3410**
4. Request code reissue if necessary

### Cannot Log Into Gateway

**Solutions:**
- **Forgot User ID:** Use account recovery (you'll need UTR and registration details)
- **Forgot Password:** Use password reset (requires User ID and 2FA access)
- **Account Locked:** Contact HMRC after multiple failed login attempts
